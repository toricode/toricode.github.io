Python abs() Function Code Examples

abs() is a Python built-in function to return the absolute value of a number.

abs() with negative interger number argument

x = abs(-5)
print(x)
Output:

5

abs() with positive interger number argument

x = abs(5)
print(x)
Output:

5

abs() with 0 in argument

x = abs(0)
print(x)
Output:

0

abs() with negative float number argument

x = abs(-9.5)
print(x)
Output:

9.5

abs() with positive float number argument

x = abs(9.5)
print(x)
Output:

9.5
