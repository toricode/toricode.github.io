Python all() Function Code Examples

all() is a Python built-in function to return True if all elements of the iterable argument are True or the iterable argument is empty.

all() with list of all True values

data_list = [True, True, True]
x = all(data_list)
print(x)
Output:

True

all() with list of not all True values

data_list = [False, True, True]
x = all(data_list)
print(x)
Output:

False

all() with list of all False values

data_list = [False, False, False]
x = all(data_list)
print(x)
Output:

False

all() with an empty list

data_list = []
x = all(data_list)
print(x)
Output:

True

all() with a set of all True values

data_set = {True, True, True}
x = all(data_set)
print(x)
Output:

True

all() with a set of not all True values

data_set = {True, False, True}
x = all(data_set)
print(x)
Output:

False

all() with a set of all False values

data_set = {False, False, False}
x = all(data_set)
print(x)
Output:

False

all() with an empty set

data_set = {}
x = all(data_set)
print(x)
Output:

True

all() with a tuple of all True values

data_tuple = (True, True, True)
x = all(data_tuple)
print(x)
Output:

True

all() with a tuple of not all True values

data_tuple = (True, True, False)
x = all(data_tuple)
print(x)
Output:

False

all() with a tuple of all False values

data_tuple = (False, False, False)
x = all(data_tuple)
print(x)
Output:

False

all() with an empty tuple

data_tuple = ()
x = all(data_tuple)
print(x)
Output:

True
